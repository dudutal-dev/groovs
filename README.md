## 🚀 העלאה לגיטהאב Pages

### שלב 1 — צור ריפוזיטורי חדש
```bash
# ב-GitHub: New Repository → שם: grooves → Public
```

### שלב 2 — העלה את הקבצים
```bash
cd /path/to/grooves
git init
git add .
git commit -m "🎵 Initial commit — Grooves vinyl app"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/grooves.git
git push -u origin main
```

### שלב 3 — הפעל GitHub Pages
```
Settings → Pages → Source: Deploy from branch → Branch: main → / (root) → Save
```

### שלב 4 — גש לאפליקציה
```
https://YOUR-USERNAME.github.io/grooves
```
