## 🚀 פריסה מהירה (GitHub Pages)

### שלב 1: Fork או העלאה
```
1. לחץ על "Fork" (למעלה מימין) כדי ליצור עותק בחשבון שלך
   - או -
   צור Repository חדש והעלה את הקבצים
```

### שלב 2: הפעל GitHub Pages
```
Settings → Pages → Source: GitHub Actions
```
זהו! האתר יהיה זמין בדקה תחת:
**`https://[שם-המשתמש].github.io/[שם-הrepo]`**

## 🔑 הגדרת מפתח API

לאחר הפתיחה:
1. עבור ל **⚙️ הגדרות**
2. הזן מפתח API מ-[console.anthropic.com](https://console.anthropic.com)
3. לחץ **שמור** → **בדוק חיבור**

> **Anthropic נותנת $5 קרדיט חינמי לחשבון חדש** — מספיק להמון שיחות.

## 🏗️ ארכיטקטורה טכנית

```
dr-claude/
├── index.html          ← האפליקציה כולה (HTML/CSS/JS)
├── README.md           ← מסמך זה
└── .github/
    └── workflows/
        └── deploy.yml  ← פריסה אוטומטית ל-GitHub Pages
```

- **Frontend:** Vanilla HTML/CSS/JavaScript (ללא תלויות)
- **AI:** Anthropic Claude Haiku 4.5 API
- **אחסון:** localStorage (מקומי על המכשיר בלבד)
- **PDF:** PDF.js (CDN)
- **פונטים:** Rubik מ-Google Fonts
